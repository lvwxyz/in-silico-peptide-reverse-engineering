# [1] 시스템 초기 아키텍처 및 하이브리드 파이프라인 구상

## 📌 작업 목표

대식세포의 면역 거부 반응(FBR)을 회피할 수 있는 생체재료의 조건을 역설계하기 위한 모델의 전체적인 수학적·구조적 아키텍처를 정의.

## 🔧 주요 변경 사항 / 시도한 접근법

- 펩타이드 서열의 분자생물학적 모티프와 양방향 문맥 파악에 특화된 1D-CNN + Bi-LSTM 융합 아키텍처 채택.
    
- 생체재료와 대식세포 간의 물리적 상호작용 및 기하학적 군집(2D/3D 공간 데이터) 연산을 위해 그래프 신경망(GNN)을 추가 도입.
    
- Flexynesis 프레임워크 기반의 변이형 오토인코더(VAE)를 적용하여 1D 서열 및 공간 전사체 데이터들을 정해진 확률 분포 상의 1D 잠재 벡터(Latent Space)로 차원 축소 및 융합하는 로직 구성.
    

## ✅ 이 단계의 결과 / 다음 단계로 넘어간 이유

이질적인 데이터를 학습하여 염증을 예측하는 [AI 엔진]과, 해당 엔진을 비용 함수로 삼아 최적해를 찾는 [유전 알고리즘 역설계 루프]의 이원화 구조가 확립되어 실제 데이터 파이프라인 전처리 단계로 진입함.

# [2] 학습 데이터 라벨링 및 15-mer Sliding Window 경량화

## 📌 작업 목표

방대한 IEDB 원본 데이터와 FASTA 천연 단백질 서열을 AI 모델이 직접적으로 학습할 수 있는 15-mer 기반 1D 텐서 형태로 정제 및 가공.

## 🔧 주요 변경 사항 / 시도한 접근법

- **IEDB 데이터 전처리:** 165개에 달하는 불필요한 메타데이터 컬럼을 `Sequence`와 `Target` 2개로 압축.
    
- **임계값 이진 분류(Thresholding):** 정답지가 없는 Assay 데이터의 경우, 결과가 `Positive`이거나 IC50 결합력이 `500nM` 이하인 서열은 `Target: 1(염증)`, `Negative`는 `Target: 0(안전)`으로 라벨링.
    
- **FASTA 안전 서열 증강:** 인간 유래 Albumin, IgG1 등 절대 안전 기준 서열(Target 0)에 15-mer Sliding Window 기법을 적용하여 15글자 단위로 분할 및 중복 제거.
    

## 💻 핵심 코드 스니펫

Python

```
def extract_sliding_window_kmers(sequence, window_size=15):
    # 긴 서열을 15글자 단위로 밀어가며 추출 (경량화 및 밀도 극대화)
    kmers = []
    if len(sequence) >= window_size:
        for i in range(len(sequence) - window_size + 1):
            kmers.append(sequence[i:i + window_size])
    return list(set(kmers))
```

## ❌ 실패하거나 버려진 시도

- 16개 아미노산 이상의 긴 서열 데이터를 통째로 사용하려 했으나, 이후 3D 구조 분석(AlphaFold) 단계 및 Tensor 메모리 제한 문제로 인해 15-mer 최대 길이 제한을 엄격하게 적용함.
    

## ✅ 이 단계의 결과 / 다음 단계로 넘어간 이유

안전 서열과 면역 유발 서열의 1D 훈련 교재가 확보되었으며, 이를 물리적인 3D 노출도(SASA) 정보와 결합하기 위해 구조 예측 연산 단계로 넘어감.

# [3] 3D 연산 병목 해결 및 SASA 대리 모델(Surrogate AI) 구축

## 📌 작업 목표

짧은 서열 예측 시 발생하는 AlphaFold2(ColabFold)의 연산 거부(Silent Fail) 에러를 우회하고, 유전 알고리즘 탐색 속도의 극심한 병목을 해소.

## 🔧 주요 변경 사항 / 시도한 접근법

- **15-mer 제한 우회:** 서열 길이가 너무 짧아 mmseqs2 서버가 3D 연산을 거부하는 한계점 극복을 위해, 유전 알고리즘(GA)의 각 세대마다 직접 AlphaFold를 호출하는 기존 루프를 전면 폐기.
    
- **Surrogate AI 도입:** 기존에 수집한 3D 데이터셋을 활용해 '1D Sequence-to-SASA 대리 예측 신경망(Surrogate Model)' 서브 모델을 학습.
    
- 대리 모델이 밀리초(ms) 단위로 15차원 SASA(아미노산 표면 노출도)를 빠르게 근사한 후, 이를 메인 멀티모달 AI(Bi-LSTM+GNN)가 받아 최종 평가하는 듀얼 AI 아키텍처 완성.
    

## ✅ 이 단계의 결과 / 다음 단계로 넘어간 이유

대규모 3D 연산을 가벼운 신경망으로 대체함으로써 역설계 탐색 속도를 약 10,000배 이상 가속화(수십 초 이내 도출)하였고, 이를 바탕으로 모델의 내부 가중치 훈련 안정화 단계로 전환함.

# [4] 데이터 불균형 극복 및 하이브리드 모델 훈련 안정화

## 📌 작업 목표

위험(염증) 데이터가 대다수로 쏠려있는 극단적 데이터 편향성을 교정하고, 학습 시 발생하는 손실 함수의 수치적 불안정성(Cascading Errors)을 제어.

## 🔧 주요 변경 사항 / 시도한 접근법

- **데이터 밸런싱(Undersampling):** AI가 모든 예측을 '1'로 찍어버리는 정확도의 역설 방지를 위해 클래스 비율을 1:2 수준으로 강제 재조정.
    
- **SASA 정규화(Normalization):** SASA 스케일(약 150 이상)이 아미노산 인코딩 수치에 비해 너무 커서 가중치가 망가지는 것을 막기 위해 값을 200으로 나누어 정규화.
    
- **손실 함수 및 스케줄러 교체:** 출력단 Sigmoid가 수반되는 `BCELoss` 대신 수치적으로 안정적인 `BCEWithLogitsLoss`로 전면 교체. Val Loss 정체 시 학습률을 조정하는 `ReduceLROnPlateau` 도입.
    

## 💻 핵심 코드 스니펫

Python

```
# 수치적 안정성을 극대화하기 위한 로직 교체
criterion = nn.BCEWithLogitsLoss()

# 3D 구조적 노출도 스케일 정규화
normalized_sasa = sasa_tensor / 200.0 
```

## ✅ 이 단계의 결과 / 다음 단계로 넘어간 이유

0으로 떨어지던 F1-Score와 AUROC 수치가 안정적으로 상승하여, 신뢰할 수 있는 예측 엔진이 완성됨. 이를 '적합도 심사위원'으로 활용해 최종 서열을 창조하는 역설계 루프의 구축으로 이어짐.

# [5] 적응형 유전 알고리즘(GA) 및 다목적 비용 함수 최적화

## 📌 작업 목표

단순 염증 회피를 넘어 물리화학적 제약 조건(수용성, 전하 등)을 모두 만족시키는 최적의 펩타이드를 도출하고, 진화 알고리즘의 조기 수렴을 방지.

## 🔧 주요 변경 사항 / 시도한 접근법

- **공학적 다목적 비용 함수 수식화:** `Cost = w1 * P_FBR + w2 * |GRAVY| + w3 * |NetCharge| - w4 * Homology_CD47 + P_constraint`
    
- **Warm-start 메커니즘:** 대식세포 식세포 억제 신호('Don't eat me')인 CD47 핵심 모티프(`EVTELT`)를 포함한 채로 초기 개체군을 생성하도록 유도.
    
- **적응형 돌연변이(Adaptive Mutation):** 세대 간 개체군의 다양성이 15% 미만으로 떨어질 경우 돌연변이율을 최대 50%까지 동적으로 상향시켜 지역 최적해(Local Optimum) 탈출 로직 구현.
    
- **Fitness Landscape Logging:** 0, 10, 20, 30세대 별 염증 점수 및 GRAVY 수치의 진화 궤적을 캡처하여 시스템 최적화 로그 추적.
    

## ❌ 실패하거나 버려진 시도

- **과도한 선택압(Selection Pressure):** 초기 세대 평가 시, 상위 1등 개체에게만 교배 기회를 몰아주었더니 단 10세대 만에 50개 펩타이드 서열이 완전히 획일화(조기 수렴)되는 버그 발생. $\rightarrow$ 이후 토너먼트 방식의 선택압을 완화하는 파레토 프론트(Pareto Front) 기법을 적용하여 해결.
    

## ✅ 이 단계의 결과 / 다음 단계로 넘어간 이유

CD47 모티프를 완벽히 보존하면서도 염증 발생 확률을 36% 이하 수준으로 낮추고 생리적 수용성 밸런스를 맞춘 최종 펩타이드 최적해 도출에 성공함. 프로젝트의 핵심 파이프라인 개발 완료.
